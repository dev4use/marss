Un produit par version

Chaque gamme de produit a une approche différente, avec sa vision produit, sa philosophie.

Un produit,

  • robuste et sans effort utilisateur, pour un usage individuel;
  • ouvert à de multiples sources et tenant la charge, pour un usage collectif;
  • totalement ouvert sur le cloud, pour des pratiques modernes.

Version STANDARD

Vous prenez des notes en markdown dans un dossier. Au moins, vous savez où sont ces notes.
Mais vous avez déjà beaucoup de notes et vous avez du mal à vous faire une vue d'ensemble.
Prenez du recul et naviguez facilement sur une interface web de site statique.
En local(A), vous pourrez consulter vos notes, classées par catégories(B).

(A) En local pour commencer, mais rien ne vous empêche de pousser les données sur GithubPages...
(B) Si vous avez l'habitude de préfixer vos fichiers d'une categorie CATEGORIE-.

Version PREMIUM

Vous utilisez déjà des markdowns un peu partout, éparpillés dans des dossiers projet sur lesquels vous travaillez ?
Pas de panique. Tout n'est pas perdu.
Nous rapatrions et traitons vos dossiers pour les publier en html.
Et le site web local est costaud, en https, avec plusieurs backends(A), pour en faire profiter vos collègues.

(A) même si la partie ops ne doit pas devenir notre préoccupatiuon principale

Version OPTIMUM

Vous êtes bien outillés, voire trop, avec tous ces outils clouds où vous prenez des notes.
Nous rapatrions toutes ces données en un seul endroit(A).
Et vous maîtrisez la publication de ces données selon le statut que vous leur donnez.
Et si vous ne jurez que par le cloud, tant mieux, c'est du standard. Nous vous aidons à remettre ces données transformée sur le cloud dans un site statique populaire typ githubPages gitlabPages... Et vous pouvez facilement changer de thème d'affichage pour suivre la marche du temps ou pour l'adapter aux codes de votre société.

(A) Un module externe tiré d'un autre de nos projets
