Dev4use

Et si pour une fois je faisais des outils à utiliser...
Parce que le dev n'est parfois qu'un prétexte à essayer quelque chose.
Et une fois que c'est fait, on passe à autre chose.
Je ne parle pas que pour moi, j'ai connu des devs passionnés qui savaient orienter le patron vers des technos qu'ils voulaient essayer...

Ce site est donc généré par MARSS (l'outil) ( ≖‿ ≖ ).
Et ce site évoque principalement le développement réalisé sur MARSS (même si je reste un peu dans la lune).
Le traditionnel README du projet MARSS vous en dira plus.

Note aux utilisateurs :
Vous pouvez indiquer votre propre page d'accueil dans la configuration du logiciel.
Cette page doit être en ".txt" afin de ne pas s'ajouter au plan de site.
